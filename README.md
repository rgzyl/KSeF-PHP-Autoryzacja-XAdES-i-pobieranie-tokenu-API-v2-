# KSeF-PHP-Autoryzacja-XAdES-i-pobieranie-tokenu-API-v2-
Kompletny przykład integracji z KSeF w PHP. Skrypt generuje podpis XAdES, pobiera challenge, podpisuje XML algorytmem ECDSA, wysyła go do API v2 i odbiera authenticationToken. Kompatybilny z środowiskami TEST / DEMO / PROD z użyciem certyfikatów autoryzacyjnych KSeF.
